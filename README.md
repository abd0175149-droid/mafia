# 🎭 Phygital Mafia Engine (محرك المافيا الهجين)

## 📌 الرؤية والنظرة العامة
نظام متطور لإدارة ألعاب "المافيا" الهجينة (Phygital Games)، يدمج بين متعة التواجد الفعلي (كروت ملموسة، لغة جسد، وتواصل بصري) والإدارة الرقمية اللحظية. يهدف النظام إلى القضاء التام على الأخطاء البشرية لليدر، أتمتة حسابات التصويت المعقدة والتقاطعات الليلية، إدارة التحالفات والاتفاقيات، وخلق تجربة سينمائية مرئية للجمهور.

## 🏗️ المعمارية التقنية (Tech Stack)
- **الواجهات الأمامية (Frontend):** `Next.js` (React), `Tailwind CSS`, `Framer Motion` (للأنيميشن).
- **محرك المزامنة (Backend):** `Node.js`, `Socket.IO` (لنقل البيانات والعدادات بأجزاء من الثانية).
- **الذاكرة الحية (State Management):** `Redis` (المصدر الوحيد للحقيقة، يحفظ الحالة اللحظية للعبة).
- **البيانات الدائمة (Database):** `PostgreSQL` + `Drizzle ORM` (لحفظ حسابات جوجل والإحصائيات).
- **الاستضافة والنشر (DevOps):** `Docker Compose`, `Cloudflare Tunnels`, `Bash Scripts`.
