# 🎨 النظام التصميمي الموحد — Phygital Noir Theme

## 1. الفلسفة البصرية (Visual Philosophy)
يعتمد تصميم النظام على ثيم **"Phygital Noir"** (الغموض المظلم المدمج بالتقنية). الهدف هو دمج الغموض الكلاسيكي والتوتر النفسي للمافيا مع الواجهات الرقمية العصرية. 
- **الاتساق المطلق:** توحيد المكونات عبر جميع الواجهات لضمان عدم تشتت المستخدم (توحيد مسار `join` مع `player`).
- **التباين العالي (High Contrast):** استخدام خلفيات داكنة جداً مع نصوص وإضاءات ساطعة (Neon/Glow) لتوجيه عين المستخدم دون الحاجة لشرح.
- **التغذية الراجعة متعددة الحواس:** كل حركة حاسمة يرافقها مؤثر بصري (نبض، اهتزاز، تدرج لوني) ومؤثر صوتي.

## 2. لوحة الألوان (Color Palette)
تم بناء لوحة الألوان للتأثير النفسي المباشر:
* **الخلفيات الأساسية:** أسود فاحم (Void Black `#050505`) مع تدرجات رمادية داكنة جداً (`#171717`) لإخفاء حواف الشاشات في الغرفة المظلمة.
* **المافيا 🔴 (Syndicate):** أحمر دموي (`#8A0303`). يتحول إلى توهج ناري (`#B22222`) في لحظات الاغتيال والإقصاء.
* **المواطنون 🔵 (Citizens):** أزرق ليلي معدني (`#1E3A5F`) وفضي للعامة (`#708090`).
* **السلطة والتنبيه 🟡:** ذهبي عتيق (`#C5A059`) للشريف وعدادات الانتظار.
* **النجاة 🟢:** أخضر نباتي (`#2E5C31`) للأحياء وتدخلات الطبيب.
* **التمييز الجندري 🟣:** أرجواني عميق (Deep Purple) لتمييز اللاعبات الإناث بمهارة في واجهات الليدر لتسهيل توزيع الأدوار.

## 3. التيبوغرافيا (Typography)
* **العناوين الرئيسية (شاشة العرض):** خطوط ضخمة جداً (7xl إلى 9xl) وحادة، تعطي طابعاً درامياً سينمائياً (ينصح بخطوط ExtraBold).
* **الأرقام والعدادات:** خطوط (Monospace) مع تأثير النبض البصري للعدادات التنازلية لزيادة التوتر.
* **نصوص القراءة:** خطوط Sans-serif حديثة ونظيفة لتسهيل القراءة في شاشات الليدر والجوال.

## 4. المؤثرات الصوتية (Web Audio API)
يتم توليد الأصوات برمجياً أو عبر ملفات مسبقة التحميل لضمان التزامن مع الأنيميشن (خاصة في شاشة العرض):
* `Tick-Tock`: مع كل صوت يتم إضافته في مرحلة التصويت.
* `Buzzer / Impact`: عند تغير ترتيب المرشحين للإقصاء.
* `Heartbeat`: دقات قلب تتسارع في مراحل التبرير والانتظار.
* `Swoosh / Slam`: عند قلب البطاقات للكشف عن الهوية.
* `Orchestration / Synth`: موسيقى تصويرية في شاشة Game Over (أصوات حادة للمافيا، ومريحة للمواطنين).
